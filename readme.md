# 香港中文大学 吕荣聪教授团队 RA 招聘

- 招聘岗位： 研究助理 (Research Assistant) 一名

- 研究方向： 智能体 (Agent) 系统研发 / 可靠软件工程

- 工作地点： 香港中文大学-华为联合实验室（深圳）

- 薪资范围： 薪资面议 (HKD)

🌟 课题组介绍

导师吕荣聪教授 (Prof. Michael R. Lyu) 是香港中文大学计算机科学与工程系教授。他是 IEEE Fellow, AAAS Fellow, ACM Fellow 及 HKAES Fellow。他曾荣获 IEEE 可靠性学会年度工程师、中国计算机学会(CCF)海外杰出贡献奖及 RGC 高级研究学者等多项荣誉。

研究领域包括软件工程、可靠计算、分布式系统、云计算、大数据及机器学习。

团队聚焦 AI 核心技术突破，主打 LLM、Agent、RAG、AI for Software Engineering (AI4SE) 等技术研究和落地。

💻 岗位职责

- 主导/参与基于强化学习 (RL) 的智能体 (Agent) 系统全流程研发，探索其在复杂软件工程（如智能运维）任务中的应用。

- 凝练科研成果，撰写人工智能/软件工程领域顶会论文 (ICSE/FSE/ASE/TSE等)。

- 深入理解强化学习基本原理与主流算法

科研素养：

综合素质： 英语读写流利，责任心强，具备团队协作与跨领域沟通能力。
